# Sprint7_tripleten

El objetivo de este proyecto es practicar las tareas habituales de la ingeniería de software.
Practicar la creación y gestión de entornos virtuales de Python y el desarrollo de una aplicación web.

Se utiliza los paquetes de pandas, plotly y streamlit para despliegue en una pagina web.
